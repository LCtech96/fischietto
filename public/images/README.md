# Cartella Immagini

Immagini per le 4 card della home page.

## File attuali:

- `Fischio-sport.png` - Immagine per la card Sport
- `Fischio-moda.png` - Immagine per la card Moda
- `Fischio-food.png` - Immagine per la card Food
- `Fischio-tech.png` - Immagine per la card Tech

## Formati supportati:

- JPG/JPEG
- PNG
- WebP

## Dimensioni consigliate:

- Formato quadrato (1:1) per un aspetto ottimale
- Dimensione minima: 800x800px
- Dimensione consigliata: 1200x1200px o superiore

Le immagini verranno automaticamente ridimensionate e ottimizzate da Next.js.
